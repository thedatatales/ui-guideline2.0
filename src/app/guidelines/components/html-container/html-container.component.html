HTML Container Component

The HTML container component renders HTML content with optional "show more/less" functionality. It supports custom styling, content truncation, and flexible toggle button positioning.

Component Properties

Property Type Default Description
content string '' HTML content string to be rendered in the container
contentClass boolean false Apply custom content class styling
showMoreOption boolean false Enable show more/less functionality
color string 'var(--grayColor600)' Custom color for the container content
showHeight string - Maximum height before showing "show more" option
linesToShow number 2 Number of lines to display before truncating
showMoreText string 'Show More' Text for the "show more" button
showLessText string 'Show Less' Text for the "show less" button
togglePosition string 'block-start' Position of toggle button: 'block-start', 'block-end', or 'inline-end'
hideToggle boolean false Hide the toggle button

Usage Guidelines

  • HTML Content: Provide HTML content as a string. Ensure content is sanitized for security
  • Show More/Less: Use showMoreOption for long content that needs truncation
  • Truncation: Use linesToShow for line-based truncation or showHeight for height-based truncation
  • Customization: Customize toggle button text and position based on design requirements
  • Styling: Use contentClass and color properties to match your application's design system
  • Security: Be cautious when rendering user-generated HTML content. Consider sanitization
  • Performance: For large HTML content, consider lazy loading or virtualization
  • Accessibility: Ensure HTML content is accessible with proper semantic markup and ARIA attributes
