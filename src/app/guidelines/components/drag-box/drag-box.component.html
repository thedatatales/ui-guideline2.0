Drag Box Component

The drag box component provides a draggable container with a label or custom template content. It displays a drag handle icon and is commonly used in form builders and drag-and-drop interfaces.

Basic Drag Box

Basic drag box displays a label with a drag handle icon. Use the label property to set the text displayed in the drag box.

Drag Box with Custom Template

Drag box can use a custom template instead of a simple label. Use the templateRef property to provide a custom template for more complex content.

Drag Box without Icon

Hide the drag handle icon by setting hideIcon to true. This is useful when you want a label-only display without the drag indicator.

Drag Box without Icon 
{{ currentCode['hideIcon'] }}

Component Properties

Property Type Default Description
label string - Text label to display in the drag box
templateRef TemplateRef<any> - Custom template for drag box content (alternative to label)
hideIcon boolean false Hide the drag handle icon

Usage Guidelines

  • Form Builders: Use drag box in form builders to create draggable form field containers
  • List Reordering: Implement drag box in lists to enable item reordering
  • Custom Templates: Use templateRef for complex content that requires formatting or multiple elements
  • Visual Indicator: The drag icon provides a visual cue that the item is draggable
  • Accessibility: Ensure drag functionality is accessible via keyboard navigation
  • Integration: Integrate with Angular CDK drag-drop for full drag-and-drop functionality
  • Styling: Customize appearance to match your application's design system
  • Performance: Use drag box efficiently in large lists with virtualization if needed
