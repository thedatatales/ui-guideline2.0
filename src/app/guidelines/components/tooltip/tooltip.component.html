Tooltip Component

The tooltip component provides contextual help and information on hover. It supports multiple directives including appTooltip and appTippy, with options for custom templates and styling.

Basic Tooltip

Basic tooltips display text on hover. Use the appTooltip directive with a string value to display simple tooltip messages.

Basic Tooltip 
{{ currentCode['basic'] }}

Tippy Tooltip

Tippy tooltips use Tippy.js library for advanced tooltip features. Use the appTippyTitle directive with various placement and styling options.

{{ currentCode['tippy'] }}

Tooltip with Custom Template

Tooltips can use custom templates for rich content. Pass a TemplateRef to the templateRef property of the tooltip directive.

Directive Properties

Property Type Default Description
appTooltip string '' Text to display in the tooltip
templateRef TemplateRef<any> - Custom template for tooltip content
offsetX number 0 Horizontal offset for tooltip position
offsetY number 50 Vertical offset for tooltip position
appTippyTitle string - Title/content for Tippy tooltip
tippyTitlePosition string 'bottom-start' Position of Tippy tooltip: 'top', 'bottom', 'left', 'right', 'top-start', 'bottom-start', etc.
tippyBgColor string 'var(--tooltipBgColor)' Background color for Tippy tooltip
tippyColor string 'var(--grayColor1000)' Text color for Tippy tooltip
tippyAllowHTML boolean false Whether to allow HTML content in Tippy tooltip
tippyDisabled boolean false Whether to disable the Tippy tooltip
tippyOptions any - Custom Tippy.js options object for advanced configuration

Usage Guidelines

  • Contextual Help: Use tooltips to provide additional information without cluttering the UI
  • Icon Explanations: Add tooltips to icon-only buttons to clarify their purpose
  • Form Fields: Use tooltips to explain form field requirements or provide examples
  • Placement: Choose appropriate placement (top, bottom, left, right) based on available space
  • Content Length: Keep tooltip text concise and informative
  • Custom Templates: Use custom templates for rich content with formatting, icons, or links
  • Tippy.js: Use Tippy tooltips for advanced features like HTML content, animations, and themes
  • Accessibility: Ensure tooltips are accessible via keyboard navigation and screen readers
