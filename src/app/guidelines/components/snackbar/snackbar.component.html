Snackbar Component

The snackbar component displays toast notifications using SnackbarService. It supports success, error, warning, and info messages with optional titles, icons, and solid styling variants.

Basic Snackbar

Basic snackbars display simple messages. Use SnackbarService methods: success(), error(), warning(), and info() to display different types of messages.

Usage Guidelines

  • Success Messages: Use for successful operations like saving data, completing actions
  • Error Messages: Use for errors, failures, or validation issues
  • Warning Messages: Use for warnings or important notices that require attention
  • Info Messages: Use for informational messages, status updates, or helpful tips
  • Duration: Default duration is 3000ms, adjust based on message length and importance
  • Position: Snackbars appear at the top-right by default, can be customized via global settings
  • Solid Type: Use solid snackbars for more prominent notifications
  • Titles and Icons: Add titles and icons for better visual communication and clarity
  • HTML Content: Enable HTML content for rich text formatting when needed
