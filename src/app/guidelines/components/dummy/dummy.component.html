Dummy Component

The dummy loader component provides a placeholder loading state for grid components. It displays a blurred skeleton UI with header controls while data is being loaded, typically used within AG Grid components.

Basic Dummy Loader

Component Properties

Property Type Default Description
asyncLoader IAsyncLoader - Configuration object for async loading state, including header options
headerTools IHeaderTools - Configuration object for header tools and controls

AsyncLoader Configuration

The asyncLoader object can contain the following configuration:

Property Type Description
headerOptions.suppressHeader boolean Suppress the header display in dummy loader
headerOptions.filterButtonPosition string Position of filter button: 'left' or 'right'
headerOptions.suppressFilterButton boolean Suppress the filter button display

Usage Guidelines

  • Grid Integration: Dummy loader is primarily used within AG Grid components during data loading
  • Loading States: Display dummy loader while fetching grid configuration or initial data
  • User Experience: Provides visual feedback that content is loading, preventing blank screens
  • Header Controls: Configure header options to match your grid's header configuration
  • Placeholder: Acts as a placeholder that mimics the final grid layout
  • Performance: Use dummy loader to improve perceived performance during data loading
  • Consistency: Ensure dummy loader configuration matches the actual grid layout
  • Accessibility: Dummy loader should indicate loading state to screen readers
