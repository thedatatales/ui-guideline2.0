Component Properties

Property Type Default Description
data any - Two-way binding for input value
label string '' Label text displayed above or beside the input
placeholder string '' Placeholder text for the input field
type string 'text' Input type: 'text', 'password', 'email', 'number', 'textbox', 'date', etc.
disabled boolean false Disable the input field
required boolean false Mark input as required
maxlength number - Maximum character length
minlength number - Minimum character length
appearance string 'outline' Form field appearance: 'outline', 'fill', 'standard'

Usage Guidelines

  1. Input Types: Choose appropriate input type based on data format (text, email, number, etc.)
  2. Placeholders: Provide clear placeholder text to guide users
  3. Validation: Use required and pattern properties for form validation
  4. Disabled State: Disable input fields when the field is not applicable
  5. Character Limits: Use maxlength and minlength for input constraints
  6. Accessibility: Ensure inputs have proper labels and ARIA attributes
  7. Error Messages: Provide clear error messages for validation failures
