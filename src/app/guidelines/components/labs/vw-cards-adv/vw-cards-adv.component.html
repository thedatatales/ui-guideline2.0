Entity → class reference

All cards use only these entities. See migration-plan/CARD_GUIDELINES.md and scss/components/_cards.scss.

EntityClass
Card container.vw-card
Header / Body / Footer.vw-card__header, .vw-card__body, .vw-card__footer
Divider.vw-card__divider
Title row (icon + title).vw-card__title-row
Title / Subtitle / Description.vw-card__title, .vw-card__subtitle, .vw-card__description
Context label.vw-card__legend
Metric (primary / secondary).vw-card__metric, .vw-card__metric--primary, .vw-card__metric--secondary
Metric row variant.vw-card__metric--row (flex row, space-between; label left, value right)
Metric label / value / row.vw-card__metric-label, .vw-card__metric-value, .vw-card__metric-row
Metric value semantic.vw-card__metric-value--info, --success, --warning, --negative (danger/error)
Metrics grid.vw-card__metrics, .vw-card__metrics--list (single column), .vw-card__metrics--divided (dividers between rows)
Header row.vw-card__header-row, .vw-card__header-row--spread (title left, value right)
Delta.vw-card__delta, .vw-card__delta--positive, .vw-card__delta--negative
Icon / Chip / Actions.vw-card__icon, .vw-card__chip, .vw-card__actions, .vw-card__action
Child card.vw-card__child.vw-card
