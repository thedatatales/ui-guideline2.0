Expansion Panel

The Expansion Panel component provides a collapsible container for displaying content. It supports single or multiple panel expansion, custom themes, and action templates.

Basic Expansion Panel

Basic expansion panel with default configuration. Only one panel can be expanded at a time. The multi property is set to false.
