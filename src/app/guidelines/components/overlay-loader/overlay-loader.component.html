Overlay Loader Component

The overlay loader component displays a loading overlay with a spinner. It can be positioned absolutely within a container or used as a full-page overlay to indicate loading states.

Basic Overlay Loader

Basic overlay loader displays a spinner with optional background color. The component covers its container and shows a loading indicator.

Property Type Default Description

position string 'absolute' CSS position property of the overlay
message string '' Primary message to display in the loader
description string '' Additional description text displayed below the message

Usage Guidelines

  • Loading States: Use overlay loader to indicate that content is being loaded or processed
  • Full Page: Use with default settings for full-page loading overlays
  • Container Overlay: Use absolute positioning within relative containers for section-specific loading
  • Background Color: Adjust background color based on content visibility needs
  • Messages: Provide clear messages to inform users what is happening
  • Conditional Display: Show/hide overlay loader based on loading state using *ngIf or @if
  • Accessibility: Ensure overlay loaders don't block keyboard navigation when appropriate
  • Performance: Use overlay loaders sparingly and remove them promptly when loading completes
