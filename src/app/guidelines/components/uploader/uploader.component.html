Uploader Component

The uploader component provides a file upload interface with drag-and-drop support, progress tracking, and file validation. It supports single and multiple file uploads with customizable styling and restrictions.

Basic Uploader

Basic uploader displays a file upload interface. Use fileChange event to get selected files and UploadBoxService to upload files to the server.

Component Properties

Property Type Default Description
acceptFile string - Accepted file extensions (e.g., '.pdf,.doc,.docx')
maxFileSize number[] - Maximum file size in bytes
fileNameLimit number 1000 Maximum length of file name
isMultiSelect boolean false Allow multiple file selection
progress number 0 Upload progress percentage (0-100)
titleText string - Title text displayed in upload box
actionText string - Action text displayed in upload box
iconName string 'cloud-arrow-up-bold' Icon name for upload box
iconColor string 'var(--grayColor400)' Color of the upload icon
boxBackgroundColor string 'var(--bgColor10)' Background color of the upload box
disabled boolean false Disable the upload box
fileChange EventEmitter<File[]> - Event emitted when files are selected

UploadBoxService Methods

Method Parameters Returns Description
uploadFilesToServer() uploadUrl: string, selectedFiles: File[] Observable Upload files to server and emit progress updates

Usage Guidelines

  • File Selection: Handle file selection using the fileChange event emitter
  • File Validation: Set acceptFile and maxFileSize to restrict file types and sizes
  • Progress Tracking: Update progress property based on upload progress from UploadBoxService
  • Drag and Drop: Component automatically supports drag-and-drop functionality
  • Multiple Files: Enable multiple file selection by setting isMultiSelect to true
  • Upload Service: Use UploadBoxService.uploadFilesToServer() to handle file uploads
  • Error Handling: Handle upload errors and display appropriate messages to users
  • File Preview: Component can display file previews for images and other supported formats
  • Customization: Customize appearance using iconColor, boxBackgroundColor, and other styling properties
