Chart Component

The chart component provides data visualization capabilities using Highcharts. It supports various chart types including column, line, pie, bar, and many more.

Usage Guidelines

  1. Chart Types: Supports various chart types: column, line, pie, bar, area, scatter, and more
  2. Options Format: Use Highcharts configuration format for chart options
  3. Observable Support: Options can be provided as Observable for dynamic updates
  4. Responsive Design: Charts automatically adapt to container size
  5. Export Functionality: Charts support export to various formats (PNG, SVG, PDF, CSV)
  6. Performance: For large datasets, consider using server-side data processing
  7. Accessibility: Ensure chart data is accessible through proper labels and descriptions
