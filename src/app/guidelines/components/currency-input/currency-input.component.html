Currency Input

The Currency Input component provides an input field with a currency selector dropdown. It supports different currency positions, custom currency lists, and formatted currency display.

Basic Currency Input

Basic currency input with default configuration. The itemList property provides the list of available currencies, and the currency selector appears on the left by default.
