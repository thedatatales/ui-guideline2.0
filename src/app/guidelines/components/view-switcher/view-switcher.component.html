View Switcher

The View Switcher (Split View) component displays a split-pane interface with a list/grid on one side and detail view on the other. It allows users to switch between different views and see details for selected items.

Basic View Switcher

Basic split view displaying a grid/list on the left side. The proxyRequired property controls whether proxy is needed for data fetching.
