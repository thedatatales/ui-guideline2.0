Rating Component

The rating component allows users to provide feedback by selecting a rating value using stars. It supports customizable number of stars, colors, and sizes.

Component Properties

Property Type Default Description
noOfStars number 5 Total number of stars to display
value number 0 Current rating value (number of filled stars)
disable boolean false Disable user interaction
color string 'var(--accentColor500)' Color of filled stars
fillColor string 'var(--bgColor400)' Color of unfilled stars
fontSize string '1.2em' Font size of the stars
theme string 'accent' Theme color: 'primary', 'accent', 'warn'
starClick EventEmitter - Event emitted when a star is clicked
ratedValue EventEmitter - Event emitted when rating value changes

Usage Guidelines

  1. Rating Scale: Use 5 stars for most cases, 10 stars for more granular feedback
  2. Visual Feedback: Provide clear visual indication of the current rating value
  3. Color Coding: Use consistent colors across your application for rating components
  4. Accessibility: Ensure rating components have proper ARIA labels and keyboard navigation
  5. Read-only Mode: Use disabled state to display ratings without allowing changes
  6. Size Consistency: Maintain consistent star sizes throughout your application
  7. User Feedback: Provide immediate visual feedback when users interact with ratings
