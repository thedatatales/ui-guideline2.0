Component Properties

Property Type Default Description
isHeader boolean true Show/hide card header
boxShadow boolean true Enable box shadow effect
padding string '1.5em' Card padding value
errorCode number - Error code to display error message
errorMessage string - Custom error message
card-title ng-content - Content selector for card title
card-subtitle ng-content - Content selector for card subtitle
card-content ng-content - Content selector for card body

Usage Guidelines

  1. Content Organization: Use cards to group related content together
  2. Headers: Include headers with titles and subtitles for better content hierarchy
  3. Visual Hierarchy: Use box shadows to create visual depth and separation
  4. Spacing: Adjust padding based on content density and design requirements
  5. Error Handling: Use errorCode and errorMessage for displaying error states
  6. Accessibility: Ensure card content is properly structured with semantic HTML
  7. Responsive Design: Cards should adapt to different screen sizes
