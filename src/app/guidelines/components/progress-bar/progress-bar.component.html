Progress Bar Component

The progress bar component displays progress indicators for operations. It supports multiple types including line, spinner, and circle progress bars with determinate, indeterminate, and buffer modes.

Line Progress Bar

Line progress bars display progress horizontally. The type property is set to 'line' and mode can be 'determinate' or 'indeterminate'.

Spinner Progress Bar

Spinner progress bars display a circular spinner. The type property is set to 'spinner' and is typically used with mode set to 'indeterminate'.

Circle Progress Bar

Circle progress bars display progress in a circular format. The type property is set to 'circle' and supports various customization options including colors, stroke width, and labels.

Buffer Progress Bar

Buffer progress bars show both progress and buffer values. The mode property is set to 'buffer' and requires both value and bufferValue properties.

Usage Guidelines

  • Determinate Mode: Use when you know the exact progress percentage (e.g., file uploads, downloads)
  • Indeterminate Mode: Use when progress cannot be determined (e.g., loading data, processing)
  • Buffer Mode: Use to show both progress and buffering status (e.g., video loading)
  • Line Type: Best for horizontal progress indicators in forms or cards
  • Spinner Type: Best for loading states and indeterminate operations
  • Circle Type: Best for compact progress display and percentage visualization
  • Accessibility: Ensure progress bars have proper ARIA labels for screen readers
  • Color Coding: Use theme colors consistently (primary for normal, accent for highlights)
