Document Preview Component

The document preview component provides a comprehensive viewer for various document types including images, PDFs, videos, audio files, spreadsheets, and text documents. It supports zooming, rotation, downloading, and navigation through multiple documents.

Basic Document Preview

Basic document preview displays a single document. Provide document data through the data property with sliderData array containing document information.

Supported Document Types

The document preview component supports various document types:

  • Images: JPG, PNG, GIF, SVG (type: 'image')
  • PDFs: PDF documents (type: 'pdf')
  • Videos: MP4, WebM, and other video formats (type: 'video')
  • Audio: MP3, MP4 audio files (type: 'audio')
  • Documents: DOCX files (type: 'docx')
  • Spreadsheets: XLSX files (type: 'xlsx')
  • Text: Plain text files (type: 'text')

Component Properties

Property Type Default Description
data any - Document data object containing sliderData array with document items
options any - Configuration options for actions, zoom, and display settings
displayNameOnHeader boolean false Display document name in the header
backgroundColor any - Background color for the preview area
loader boolean false Show loading indicator
showSide boolean false Show side panel
zoomLevel number 1 Initial zoom level for the document
documentPreviewConfig object - Configuration for document preview dimensions and padding
hideHeader boolean false Hide the preview header
selectedAction EventEmitter - Event emitted when a user action (zoom, download, etc.) is selected
onSaveRotation EventEmitter - Event emitted when image rotation is saved

Options Configuration

The options object can contain the following configuration:

Option Type Description
action.zoom boolean Enable zoom functionality
action.info boolean Show info action button
action.share boolean Show share action button
action.rotate boolean Enable rotation functionality
action.edit boolean Show edit action button
action.delete boolean Show delete action button
action.download boolean Show download action button
action.isRotationSave boolean Save rotation changes
zoomOption.zoomOnMouseWheel boolean Enable zoom on mouse wheel
arrowBtn boolean Show navigation arrows for multiple documents

Usage Guidelines

  • Document Data: Provide document data as blob objects or base64 strings depending on document type
  • Multiple Documents: Use carousel navigation for multiple documents by adding items to sliderData array
  • Action Buttons: Configure available actions through the options.action object
  • Zoom Controls: Enable zoom functionality for images and PDFs using zoomOption settings
  • Rotation: Use rotation feature for images and save rotation state with isRotationSave option
  • Event Handling: Listen to selectedAction event to handle user interactions like download, delete, etc.
  • Performance: For large files, consider lazy loading or progressive loading strategies
  • Accessibility: Ensure document previews are accessible with keyboard navigation and screen reader support
