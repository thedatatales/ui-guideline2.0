Angular Calendar

The Angular Calendar component provides a flexible calendar interface using the angular-calendar library. It supports month, week, and day views with customizable events and navigation.

Month View

Month view displays events in a monthly calendar grid. The mwl-calendar-month-view component shows all days of the month with events displayed on their respective dates.

Week View

Week view displays events in a weekly timeline format. The mwl-calendar-week-view component shows events across the days of the week with hourly segments.

Day View

Day view displays events for a single day in a timeline format. The mwl-calendar-day-view component shows events throughout the day with hourly segments.
