Page Header Component

The page header component provides a consistent header layout for pages across the application. It supports left, middle, and right content sections using content projection.

Basic Page Header

Basic page header displays content in the default left section. Use content projection to add your header content.

Page Header with Multiple Sections

Page header supports three content sections: left, middle, and right. Use the content projection selectors header-content-left, header-content-middle, and header-content-right to place content in specific sections.

Content Projection

The page header component uses Angular content projection to allow flexible content placement. Use the following selectors to target specific sections:

  • Default content: Content without a selector is placed in the left section
  • header-content-left: Content for the left section of the header
  • header-content-middle: Content for the middle section of the header
  • header-content-right: Content for the right section of the header

Usage Guidelines

  • Consistency: Use page header consistently across all pages for a unified user experience
  • Content Organization: Place primary navigation/title in left, search/filters in middle, actions in right
  • Responsive Design: Ensure header content adapts well to different screen sizes
  • Accessibility: Include proper heading hierarchy and ARIA labels for screen readers
  • Flexibility: Use content projection to customize header content based on page requirements
