Annotation Tool

The Annotation Tool component provides image annotation capabilities, allowing users to draw shapes, add text, arrows, and custom images on images. It supports capturing annotated images and exporting annotation data.

Basic Annotation Tool

Basic annotation tool with standard drawing capabilities. The imageJson property stores all annotation data including circles, rectangles, lines, arrows, and text.
