Checkbox Component

Checkboxes allow users to select one or more options from a set. The BNtv Component Library provides comprehensive checkbox components that follow Material Design principles while offering flexibility for various use cases.

All checkbox components are production-ready, accessible, and theme-aware. Use checkboxes for multiple selections where users can choose zero or more options.

Component Properties

Property Type Default Description

Usage Guidelines

  1. Single Selection: Use checkboxes for multiple selections where users can choose zero or more options
  2. Boolean Input: Use checkboxes for simple yes/no or on/off toggles
  3. Form Validation: Use the required property for form validation
  4. Indeterminate State: Use indeterminate state for "select all" scenarios where some items are selected
  5. Disabled State: Disable checkboxes when the option is not available or not applicable
  6. Label Clarity: Always provide clear, descriptive labels for checkboxes
  7. Accessibility: Ensure checkboxes have proper ARIA labels and keyboard navigation
