Classes Reference

Complete reference of all card classes available from core-mixin.scss (lines 159-305) with usage guidelines.

Class Sample Usage When to Use
.vw-card-section
Card Content
 Main card container Use as the primary wrapper for standard cards. Provides white background, border, rounded corners (16px via --vw-radius-lg), and padding (1rem via --vw-space-lg). Best for standalone cards or card grids.
.vw-card-parent
Parent Card
 Parent card container Use when creating nested card structures. Border radius 16px (via --vw-radius-lg), padding 1rem (via --vw-space-lg), gap 0.5rem (via --vw-space-sm). Ideal for cards that contain child cards or grouped content.
.vw-card-child
Child Card
 Child card container Use inside .vw-card-parent for nested card hierarchies. Padding 0.5rem (via --vw-space-sm) and border radius 8px (via --vw-radius-sm). Perfect for sub-items or related content groups.
.vw-card-child-shaded
Shaded Child
 Shaded child card container Use inside .vw-card-parent for nested card hierarchies with a subtle background. Same padding and border radius as .vw-card-child, but with slate-50 background (via --vw-color-slate-50). Perfect for alternating or highlighted sub-items.
.vw-card-title
Card Title
 Card title text Use for primary heading text within cards. Font size 1rem (16px via --vw-font-heading-md), weight 500, line height 1.25 (via --vw-line-heading-md), gray-800 color. Apply to main card titles, section headers, or primary labels.
.vw-card-activity-label
Knowledge Hub
 Card activity label text Use for activity labels or list item titles within cards. Font size 1rem (16px via --vw-font-heading-md), weight 500, line height 1.5, gray-700 color. Apply to document names, folder names, or labels that precede activity values.
.vw-card-activity-value
December 26, 2024
 Card activity value text Use for activity values like dates, file sizes, or metadata. Font size 0.875rem (14px via --vw-font-description), line height 1.4 (via --vw-line-description), gray-500 color. Apply below list titles or after activity labels.
.vw-card-description
Card description text
 Card description text Use for secondary or explanatory text. Font size 0.875rem (14px via --vw-font-description), weight 400, line height 1.4 (via --vw-line-description), gray-600 color. Ideal for subtitles, captions, or supporting information.
.vw-card-metric-xxl
₹5.0M
 Extra extra large metric value Use for the largest numeric values or hero metrics. Font size 2rem (32px), weight 400, line height 1.2, gray-800 color. Best for hero sections, main KPIs, or the most prominent data displays.
.vw-card-metric-xl
₹2.5M
 Extra large metric value Use for primary numeric values or key metrics. Font size 1.5rem (24px via --vw-font-value-xl-alt), weight 400, line height 1.2, gray-800 color. Best for KPI cards, dashboards, or prominent data displays.
.vw-card-metric-lg
₹1.8M
 Large metric value Use for large numeric values or secondary metrics. Font size 1.25rem (20px via --vw-font-value-lg), weight 400, line height 1.2, gray-800 color. Suitable for supporting data or secondary KPIs.
.vw-card-metric-md
1,250
 Medium metric value Use for medium numeric values or moderate emphasis metrics. Font size 1.125rem (18px via --vw-font-value-md), weight 400, line height 1.25, gray-800 color. Suitable for supporting data or secondary metrics.
.vw-card-metric-sm
42
 Small metric value Use for small numeric values or less prominent metrics. Font size 1rem (16px via --vw-font-value-sm), weight 400, line height 1.25, gray-800 color. Good for supplementary data or compact displays.
.vw-card-metric-label
Total Revenue
 Metric label text Use to label metric values. Font size 0.875rem (14px via --vw-font-label-md), weight 400, line height 1.2, gray-600 color. Place above or beside metric values to provide context.
.vw-card-variance
+12%
 Variance indicator Use to show change indicators (percentage, trend). Font size 0.75rem (12px via --vw-font-legend), weight 500, line height 1.2 (via --vw-line-legend). Combine with modifier classes for color coding.
.vw-card-variance.is-positive
+15.2%
 Positive variance Use for positive changes, increases, or gains. Green color (via --vw-color-green-700). Apply to upward trends, growth metrics, or favorable changes.
.vw-card-variance.is-negative
−8.5%
 Negative variance Use for negative changes, decreases, or losses. Red color (via --vw-color-red-600). Apply to downward trends, decline metrics, or unfavorable changes.
.vw-card-variance.is-neutral
0%
 Neutral variance Use for neutral changes, no change, or stable metrics. Gray color (via --vw-color-gray-500). Apply when there's no significant change or status is unchanged.
.vw-card--success
Success
 Success variant Use for success states, completed actions, or positive status. Light green background (via --vw-color-green-50) with green border (via --vw-color-green-400). Apply to success messages, completed tasks, or positive indicators.
.vw-card--error
Error
 Error variant Use for error states, failed actions, or critical issues. Light red background (via --vw-color-red-50) with red border (via --vw-color-red-300). Apply to error messages, failed operations, or critical alerts.
.vw-card--warning
Warning
 Warning variant Use for warning states, cautionary messages, or attention-needed items. Light yellow background (via --vw-color-yellow-50) with yellow border (via --vw-color-yellow-400). Apply to warnings, cautions, or items requiring attention.
.vw-card--info
Info
 Info variant Use for informational states, general messages, or neutral status. Light blue background (via --vw-color-blue-50) with blue border (via --vw-color-blue-300). Apply to informational messages, tips, or neutral status indicators.
.vw-chip Chip Base chip component Use for badges, tags, or status indicators. Padding 0.375rem 0.75rem, rounded pill shape (9999px). Combine with variant classes for different semantic meanings.
.vw-chip--success Active Success chip Use for success badges or positive status tags. Green background (via --chipSuccessBgColor or --vw-color-green-100) with green text (via --chipSuccessTextColor or --vw-color-green-700).
.vw-chip--error Failed Error chip Use for error badges or negative status tags. Red background (via --chipErrorBgColor or --vw-color-red-100) with red text (via --chipErrorTextColor or --vw-color-red-700).
.vw-chip--warning Pending Warning chip Use for warning badges or cautionary tags. Yellow background (via --chipWarningBgColor or --vw-color-yellow-100) with yellow text (via --chipWarningTextColor or --vw-color-yellow-700).
.vw-chip--info New Info chip Use for informational badges or neutral tags. Blue background (via --chipInfoBgColor or --vw-color-blue-100) with blue text (via --chipInfoTextColor or --vw-color-blue-700).
.vw-chip--neutral Default Neutral chip Use for neutral badges or default tags. Gray background (via --vw-color-gray-100) with gray text (via --vw-color-gray-600).
.vw-chip--purple Custom Purple chip Use for custom badges or special category tags. Purple background (via --vw-color-purple-100) with purple text (via --vw-color-purple-700).
.vw-chip--cyan Cyan Cyan chip Use for cyan badges or category tags. From core-mixin.scss.
.vw-chip--orange Orange Orange chip Use for orange badges or category tags. From core-mixin.scss.
.vw-chip--pink Pink Pink chip Use for pink badges or category tags. From core-mixin.scss.
.vw-chip.is-strong Strong Strong weight chip Modifier: font-weight 500. Combine with any .vw-chip--* variant.
.vw-chip.is-clickable Clickable Clickable chip Modifier: cursor pointer. Combine with any .vw-chip--* variant.
.common-red-chip Red Common red chip Legacy chip variant from core-mixin.scss. Use for red badges or tags.
.common-orange-chip Orange Common orange chip Legacy chip variant from core-mixin.scss.
.common-brown-chip Brown Common brown chip Legacy chip variant from core-mixin.scss.
.common-yellow-chip Yellow Common yellow chip Legacy chip variant from core-mixin.scss.
.common-coffee-chip Coffee Common coffee chip Legacy chip variant from core-mixin.scss.
.common-green-chip Green Common green chip Legacy chip variant from core-mixin.scss.
.common-magenta-chip Magenta Common magenta chip Legacy chip variant from core-mixin.scss.
.common-lavender-chip Lavender Common lavender chip Legacy chip variant from core-mixin.scss.

.common-lightBlue-chip Light blue Common light blue chip Legacy chip variant from core-mixin.scss.
