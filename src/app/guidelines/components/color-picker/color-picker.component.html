Component Properties

Property Type Default Description
selectedColor string '#000000' Two-way binding for selected color value (hex format)
label string '' Label text displayed above or beside the color picker
labelPosition string 'top' Label position: 'top', 'before', 'after'
border boolean false Display border around color picker
showColorCode boolean false Display color code text next to color picker
showInput boolean false Show input field for manual color entry
showIcon boolean true Show expand icon
disabled boolean false Disable the color picker
required boolean false Mark color picker as required
pickerBoxConfig object - Configuration for picker box styling (height, width, border, borderRadius)
ColorChangeUpdate EventEmitter - Event emitted when color changes
selectedColorChange EventEmitter - Event emitted for two-way binding support

Usage Guidelines

  1. Color Selection: Use color picker for visual color selection in forms and settings
  2. Color Codes: Enable showColorCode when users need to see or copy hex values
  3. Input Field: Use showInput for advanced users who prefer typing hex codes
  4. Border Styling: Use border property for better visual definition
  5. Disabled State: Disable color picker when the color option is not available
  6. Label Position: Choose appropriate label position based on layout requirements
  7. Accessibility: Ensure color picker has proper ARIA labels and keyboard navigation
