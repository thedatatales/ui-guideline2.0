vw-utility

This page helps developers understand layout and utility classes (e.g. .vw-page-gap) from core-mixin.scss. No new classes are added; we only show how to combine and use them.

1. vw-page-gap

.vw-page-gap is a utility from core-mixin.scss (lines 195â€“206) that sets gap on a flex container. It uses var(--vw-page-gap) (0.75rem in gx-theme). Use it on the flex container (not on the children) to space flex items evenly.

Variants: .vw-page-gap-xl (--vw-space-xl), .vw-page-gap-md (--vw-space-md), .vw-page-gap-sm (--vw-space-sm). These additional classes are for specific needs when you want a different gap size than the default.

Where to apply .vw-page-gap

Apply .vw-page-gap on the flex container (column or row). The diagram below shows a flex column with three blocks; the strips between them represent the gap. Hover over the gap strips to see the spaced area.
