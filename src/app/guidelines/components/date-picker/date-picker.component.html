Component Properties

Property Type Default Description
pickerType string 'both' Picker type: 'calendar', 'timer', 'both', 'month-calendar', 'year-calendar'
selectMode string 'single' Selection mode: 'single', 'range', 'rangeFrom', 'rangeTo', 'week', 'multiple'
data Date | Date[] - Two-way binding for selected date/time value(s)
label string '' Label text displayed above or beside the picker
placeholder string '' Placeholder text for the input field
pickerPosition string - Calendar popup position: 'left' or default (right)
rangeSeparator string '~' Character to separate 'from' and 'to' in range mode
hour12Timer boolean false Use 12-hour format (AM/PM) for time picker
showSecondsTimer boolean false Show seconds in time picker
hideActions boolean false Hide action buttons in picker
min Date - Minimum selectable date/time
max Date - Maximum selectable date/time
minTime string - Minimum selectable time (format: 'HH:mm:ss')
maxTime string - Maximum selectable time (format: 'HH:mm:ss')
minTimeErrorMsg string - Error message for invalid minTime
maxTimeErrorMsg string - Error message for invalid maxTime
setMaxZeroHour boolean true Set max date time to 00:00:00
timeFormat string - Custom time format (e.g., 'hh:mm A')
multiYearView boolean false Show multiple years in year picker
hideHeaderDate boolean false Hide header date display
advancedCalendar boolean false Enable advanced calendar with custom options
disabled boolean false Disable the picker
readonly boolean false Make input field readonly
required boolean false Mark picker as required

Usage Guidelines

  1. Calendar Type: Use [pickerType]="'calendar'" for date-only selection
  2. Timer Type: Use [pickerType]="'timer'" for time-only selection
  3. Both Type: Use [pickerType]="'both'" for combined date and time selection
  4. Month Calendar: Use [pickerType]="'month-calendar'" for month and year selection
  5. Year Calendar: Use [pickerType]="'year-calendar'" for year-only selection
  6. Single Selection: Use [selectMode]="'single'" for single date/time selection
  7. Range Selection: Use [selectMode]="'range'" for full range, 'rangeFrom'/'rangeTo' for partial ranges
  8. Week Selection: Use [selectMode]="'week'" for week selection
  9. Multiple Selection: Use [selectMode]="'multiple'" for selecting multiple dates
  10. Time Constraints: Use minTime and maxTime to restrict selectable time ranges
  11. Date Constraints: Use min and max to restrict selectable dates
  12. Time Format: Use hour12Timer for 12-hour format and timeFormat for custom formats
  13. Advanced Calendar: Use [advancedCalendar]="true" for enhanced calendar functionality
  14. Accessibility: Ensure pickers have proper ARIA labels and keyboard navigation
