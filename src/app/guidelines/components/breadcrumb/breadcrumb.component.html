Breadcrumb Component

The breadcrumb component displays a navigation path showing the user's location in the application hierarchy. It helps users understand where they are and navigate back to parent pages.

Component Properties

Property Type Default Description
pageInfo any - Object containing navigation array and optional subtitle
showTitle boolean false Show page title
isCommon boolean false Use common breadcrumb style
maxBreadNumber number 3 Maximum number of breadcrumb items to display
suppressBreadCrumb boolean true Suppress breadcrumb display
breadColor string 'var(--breadCrumbBreadColor)' Color for breadcrumb items
recentBreadColor string 'var(--breadCrumbRecentBreadColor)' Color for the current/last breadcrumb item
iconColor string 'var(--breadCrumbIconColor)' Color for separator icons
iconName string 'arrow-right-curve' Icon name for separator
iconSize string '1.35em' Size of separator icons
onClick EventEmitter - Event emitted when a breadcrumb item is clicked

Usage Guidelines

  1. Navigation Structure: Provide clear, hierarchical navigation paths
  2. Current Page: The last item should represent the current page (without route)
  3. Clickable Items: All items except the last should have routes for navigation
  4. Breadcrumb Length: Keep breadcrumbs concise; use maxBreadNumber for long paths
  5. Subtitle: Use subtitle to provide additional context about the current page
  6. Accessibility: Ensure breadcrumbs have proper ARIA labels and keyboard navigation
  7. Consistency: Maintain consistent breadcrumb styling across the application
