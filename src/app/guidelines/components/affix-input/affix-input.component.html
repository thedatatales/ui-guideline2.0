Affix Input

Affix Input allows adding prefix and suffix content (icons, text) to input fields for additional context or actions. This page will be expanded with full visual examples and code snippets.

Overview

Use the Affix Input component when you need to display units, icons, or helper text directly inside the input field area, aligned to the left or right of the user input.
