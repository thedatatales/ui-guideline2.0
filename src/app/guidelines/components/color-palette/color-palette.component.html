Color Palette

The color palette defines the color system used throughout the application. Column one shows vw-palette variables; column two shows theme variables (Primary, Accent, Warn, Gray) from the active theme.

Theme colors use var(--primaryColor500), var(--accentColor500), etc. and update dynamically based on the active theme (body class: theme-indigo, theme-black, or theme-greenwave).

Usage Guidelines

  • vw-palette: Use --vw-color-violet-*, --vw-color-teal-*, etc. for fixed design tokens.
  • Theme variables: Use var(--primaryColor500), var(--accentColor500), etc. so colors follow the active theme.
  • Click to copy: Click any swatch row to copy its hex value to the clipboard.
