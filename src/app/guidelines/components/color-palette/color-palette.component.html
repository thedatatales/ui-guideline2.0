Usage Guidelines

  • Primary Colors: Use primary colors for main actions, navigation bars, and key UI elements
  • Accent Colors: Use accent colors for secondary actions, links, and highlights
  • Warning Colors: Use warning colors for error messages, alerts, and destructive actions
  • Gray Scale: Use gray scale for text, borders, backgrounds, and neutral UI elements
  • Contrast: Ensure sufficient contrast between text and background colors for accessibility
  • Consistency: Use color values consistently across the application to maintain visual harmony
  • CSS Variables: Colors are available as CSS variables (e.g., var(--primaryColor500))
  • Click to Copy: Click on any color swatch to copy its hex value to the clipboard
