Toggle Component

Usage Guidelines

  1. Slide Toggle: Use for simple on/off switches and binary choices
  2. Extended Toggle: Use when you need to provide additional context with a description
  3. Button Toggle: Use for selecting from multiple related options
  4. State Management: Always use two-way binding for toggle state management
  5. Visual Feedback: Provide clear visual indication of the current toggle state
  6. Accessibility: Ensure toggles have proper labels and keyboard navigation
  7. Grouping: Use button toggles for mutually exclusive options
