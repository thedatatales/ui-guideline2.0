Inline Edit Component

The inline edit component allows users to edit content directly in place without navigating to a separate edit page. It supports various trigger modes and save behaviors.

Component Properties

Property Type Default Description
inlineEditorConfig IConfigInlineEditor - Configuration object for inline editor behavior
editable boolean true Whether the field is editable
editableOn EditOptions EditOptions.CLICK Trigger mode: EditOptions.CLICK, EditOptions.DB_CLICK, EditOptions.ICON_CLICK
saveOn SaveOnType SaveOnType.OUTSIDE_CLICK Save behavior: SaveOnType.OUTSIDE_CLICK, SaveOnType.ICON_CLICK, SaveOnType.VALUE_CHANGE
displayMode ModeType ModeType.LABEL_WRAP Display mode: ModeType.LABEL_WRAP, ModeType.EDITOR
onSave EventEmitter - Event emitted when changes are saved
onCancelInline EventEmitter - Event emitted when editing is cancelled
onEditing EventEmitter - Event emitted when editing starts

Usage Guidelines

  1. Trigger Modes: Use 'Click' for quick edits, 'DB_Click' to prevent accidental edits
  2. Save Behavior: Use 'Outside_Click' for intuitive save, 'Enter' for keyboard-friendly editing
  3. Editor Mode: Use 'text' for simple text, 'editor' for rich text editing
  4. User Feedback: Provide visual feedback when entering edit mode
  5. Validation: Validate input before saving inline edits
  6. Accessibility: Ensure inline edit has proper keyboard navigation and ARIA labels
  7. Cancel Option: Always allow users to cancel inline edits
