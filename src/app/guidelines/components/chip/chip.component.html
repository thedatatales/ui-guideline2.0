Chip

The Chip component displays compact elements that represent input, attribute, or action. Chips can be used for tags, selections, input with autocomplete, and removable items.

Basic Chip

Basic chip component displaying a list of chips. The data property contains the array of chip items, and displayExpression defines how to display each chip.
