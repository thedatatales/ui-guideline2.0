Auto Complete

The Auto Complete component provides an input field with autocomplete suggestions. It supports single and multi-select modes, API integration, and custom display expressions.

Basic Auto Complete

Basic auto complete with a static list of options. The autocompleteoptions property provides the list of suggestions, and displayExpression defines how to display each option.
