Menu Component

The menu component provides a dropdown menu interface with support for nested menus, icons, and custom templates. It is built on top of Angular Material's menu component.

Basic Menu

Basic menu displays a simple list of menu items. The list property contains the menu items, and resultAction event is emitted when a menu item is clicked.

Component Properties

Property Type Default Description
list MenuItem[] - List of all menu items
matMenu MatMenu - Reference to Angular Material menu
overlapTrigger boolean false Whether the menu should overlap its trigger
yPosition 'above' | 'below' 'below' Position of the menu in the Y axis
xPosition 'before' | 'after' 'after' Position of the menu in the X axis
dynamicMenuClass string '' Dynamic class for the menu
backdropClass string 'cdk-overlay-transparent-backdrop' Backdrop class for the menu
template TemplateRef<any> - Template for custom menu content
optionTemplate TemplateRef<any> - Template for custom menu item rendering
resultAction EventEmitter<string> - Event emitted when a menu item is clicked

Usage Guidelines

  • Menu Triggers: Use Angular Material's matMenuTriggerFor directive to attach menu to buttons or other elements
  • Menu Items: Define menu items with title, action, and optional icon properties
  • Nested Menus: Create nested menus by adding child arrays to menu items
  • Icons: Add icons to menu items using icon, iconType, and fontSet properties
  • Custom Templates: Use optionTemplate to customize the appearance of menu items
  • Positioning: Control menu position using yPosition and xPosition properties
  • Actions: Handle menu item clicks using the resultAction event emitter
  • Disabled Items: Set disabled property on menu items to disable them
