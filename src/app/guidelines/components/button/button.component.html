Button Component

Buttons are interactive elements that allow users to trigger actions and make choices with a single tap or click. The BNtv Component Library provides comprehensive button components that follow Material Design principles while offering flexibility for various use cases.

All button components are production-ready, accessible, and theme-aware. Choose button types based on their purpose and visual hierarchy, not personal preference.

Usage Guidelines

  1. Primary Actions: Use raised or stroked buttons for primary actions
  2. Secondary Actions: Use flat or flat-stroked buttons for secondary actions
  3. Icon Buttons: Use icon or icon-stroked buttons for icon-only actions
  4. Floating Actions: Use fab or mini-fab for floating action buttons
  5. Themes: Use 'primary' for main actions, 'accent' for alternative actions, 'warn' for destructive actions
  6. Disabled State: Always provide visual feedback for disabled buttons
  7. Accessibility: Ensure buttons have proper ARIA labels and keyboard navigation
