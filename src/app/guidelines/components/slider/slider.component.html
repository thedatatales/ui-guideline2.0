Component Properties

Property Type Default Description
label string - Label text for the slider
min number 0 Minimum value of the slider
max number 100 Maximum value of the slider
value number 0 Current value - use two-way binding [(value)]
step number 1 Step increment value
disabled boolean false Disable the slider
thumbLabel boolean false Show thumb label on hover
showThumbLabelAlways boolean false Always show thumb label
tickInterval number 0 Interval for displaying tick marks
showCountBox boolean false Show input box for value entry
boxPosition string 'topRight' Position of input box: 'topLeft', 'topRight', 'bottomLeft', 'bottomRight'
theme string 'accent' Theme color: 'primary', 'accent', 'warn'
type string 'default' Slider type: 'default', 'mat', 'opacitySlider'
showMinMaxValues boolean false Show min and max values on the slider
valueChange EventEmitter - Event emitted when slider value changes

Usage Guidelines

  1. Range Selection: Use sliders for selecting values within a continuous range
  2. Step Values: Set appropriate step values based on the use case (e.g., 1 for integers, 0.1 for decimals)
  3. Input Box: Use showCountBox for scenarios requiring precise value entry
  4. Visual Feedback: Use thumbLabel or showThumbLabelAlways to provide immediate value feedback
  5. Accessibility: Ensure sliders have proper labels and keyboard navigation support
  6. Default Values: Set sensible default values that fall within the min-max range
  7. Performance: For real-time updates, consider debouncing the valueChange event
