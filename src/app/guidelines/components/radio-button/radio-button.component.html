Radio Button Component

The radio button component allows users to select a single option from a group of options. It supports single radio buttons and radio button groups with various themes and states.

Usage Guidelines

  1. Single Selection: Use radio buttons when users must select exactly one option from a set
  2. Group vs Single: Use 'group' for multiple options, 'single' for boolean/toggle scenarios
  3. Layout Direction: Use 'row' for horizontal layout, 'column' for vertical layout
  4. Display Expression: Use displayExpression to format option display (e.g., '$name ($code)')
  5. Disabled State: Disable radio buttons when options are not available
  6. Required Fields: Mark radio groups as required for form validation
  7. Accessibility: Ensure radio buttons have proper labels and keyboard navigation
