Calendar

The Calendar component provides a full-featured calendar interface built on FullCalendar. It supports multiple views (month, week, day), event management, custom templates, and integration with external calendar providers like Google Calendar.

Basic Calendar

Basic calendar with default configuration. The calendarConfig property controls the calendar's behavior, including header options, side view, and event settings.
