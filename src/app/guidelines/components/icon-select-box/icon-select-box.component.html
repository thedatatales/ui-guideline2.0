Icon Select Box

The Icon Select Box component provides a dropdown interface for selecting icons or images. It supports different display modes, custom sizing, and can display icons with or without names.

Basic Icon Select Box

Basic icon select box displaying icons only. The displayMode property is set to 'onlyIcon' to show only icons without text labels.
